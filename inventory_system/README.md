# Inventory Management System

## Project Overview
This is a Django-based Inventory Management System that allows users to add and view products.

## Features
- Add products
- View inventory list
- Django admin support
- SQLite database

## Tech Stack
- Python
- Django
- HTML/CSS
- SQLite

## Setup Instructions

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
   pip install django
4. Run migrations
   python manage.py migrate
5. Start server
   python manage.py runserver

## Environment Variables
Refer to `.env.example` for required environment variables.
