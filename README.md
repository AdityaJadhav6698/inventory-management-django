# Inventory Management System (Django)

## Project Overview
This project is a Django-based Inventory Management System that allows users to add products and view inventory details.

## Features
- Add new products
- View inventory list
- Update product stock
- Django admin support

## Technologies Used
- Python
- Django
- HTML/CSS
- SQLite

## Project Structure
- inventory/ : Django app
- inventory_system/ : Project settings
- manage.py : Project entry point
- db.sqlite3 : Database file

## Setup Instructions
1. Clone the repository
2. Install Django
   pip install django
3. Apply migrations
   python manage.py migrate
4. Run the server
   python manage.py runserver

## Environment Variables
Refer to the `.env.example` file for required environment variables.

## Run the Application
Open browser and go to:
http://127.0.0.1:8000/
